This add-on enables you to easily share tags with other Anki users. It exports all notes that have a chosen tag. All other tags will not be present on the exported notes. To export a tag go to the AnKing menu -> Export Tag Receivers of the tag can import it using the 'Import Tag' feature of the Special Fields add-on Screenshots Problems, Bugs, Errors, Improvements If you find a serious bug, please submit it on Github or email us at ankingmed@gmail.com.
